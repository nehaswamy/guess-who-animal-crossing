Guess Who? Animal Crossing Edition

Rules

  • Each player has a set of 24 different villagers. Each player should have the same board. Players can choose which set they want by typing in a number from 1 through 16.
  • A villager will be chosen at random. The background color will be changed to gray to indicate this.
  • The youngest player goes first, beginning by asking the other player a characteristic about the villager.
  • If they say, “Yes,” the asking player clicks on all of the characters without that characteristic. If they say, “No,” the asking player clicks on all the villagers with that characteristic.
  • Players will eventually be able to “guess” the name of the opponents character."
